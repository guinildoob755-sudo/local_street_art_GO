## Introduction

--

Project Overview

As part of my work as an independent developer, I was contacted by Lisa, director of the ColorTown association, to participate in the development of an innovative mobile application.

ColorTown is an organization dedicated to promoting street art and graffiti artists. It aims to implement a digital solution to map and archive urban artworks on a global scale.

Within this context, the Local StreetArt project was created: a collaborative mobile application allowing users to photograph, share, and discover street art pieces.

The goal is to build a living database of often ephemeral artworks, accessible to everyone.

Following the acquisition of public funding, the project has entered the development phase.

Reformulation of Client Requirements

The ColorTown association wants to develop a mobile application called LocalStreetArt, allowing users to list and share street art pieces worldwide.

The application enables users to:

create an account and log in
upload photographs of urban artworks
browse published artworks
like posts
take photos directly using the phone camera

Each post contains:

an image
the username
a like counter

The application is developed using React Native (Expo) and uses Firebase for authentication, storage, and database management.

The goal is to deliver a simple, modern, and intuitive application that promotes cultural sharing.
